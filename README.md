# compiler
2022 Compile-Principle Project<br>
A tiny C compiler (maybe)
## Env
Dependence:
```
flex 2.6.4
bison 3.5.1
llvm 10.0.0
```
Tested in Ubuntu 20.04 for x86-64
## Build
For Linux:
```
sudo apt install flex bison llvm
make
```
